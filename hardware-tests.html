Bitcoin Mining Hardware Test Table

This table documents the Bitcoin mining hardware used to test the Hydra Pool Stratum implementation. This is not an exhaustive list of all Bitcoin mining hardware but best efforts have been made to make it comprehensive.
  • The "Make" column defines the manufacturer.
  • The "Model" column indicates the specific piece of hardware.
  • The "Firmware" column specifies the firmware version.
  • Green cells mean it was tested with success.
  • Yellow cells mean it has not been tested and we need your help.
  • Red cells mean it was tested but failed.

Testing

If you have a piece of hardware from any of the yellow cells, then please test it by pointing your miner to this URL: stratum+tcp://test.hydrapool.org:3333

The server logs will be monitored for test results and the table will be updated accordingly.
